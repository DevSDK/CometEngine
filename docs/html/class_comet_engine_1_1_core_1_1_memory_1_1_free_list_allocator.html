상세한 설명

Free-List를 기반으로 하는 커스텀 메모리 할당자이다.

할당된 메모리 블록을 가지고, Free-List를 구성하여, 최초 적합 메모리 할당 위치를 찾아 할당해주는 역할을 한다.
